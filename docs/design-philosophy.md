# NGデザイン(絶対避ける)

## 1. ロゴを大きく置く

ロゴ主張は安っぽく見える。

### NG例
- 左上巨大ロゴ
- 背景写真にベタ置き
- 金ロゴを全面表示

### 理想
- 小さく
- 余白多く
- 溶け込ませる

---

## 2. "美容室感" を出しすぎる

禁止：
- ピンク
- キラキラ
- 強いグラデ
- 韓国風UI
- インスタ広告感

Salon Rink は
「AI SaaS × 高級ホテル」
である。

---

## 3. 文字を長くしない

NG：
「初回来店後の満足度は高いのに〜」

### 理想
短く、強く。

例：

# 「再来店は、"感覚"では増えない。」

# 「AIが、次回来店を設計する。」
