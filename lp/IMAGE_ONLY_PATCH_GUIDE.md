# SalonRink.com — 画像差替/新設 指示書

> **スコープ**: テキスト・レイアウト・配列・文章は**一切変更しない**。
> **画像（写真・イラスト・アイコン）の張替えまたは新設のみ**を行う。
> 各ページごとにオーナーから写真を受領し、Claude Code に対し1ページずつ差分指示を発行する。

---

## 厳守ルール（Claude Code 向け）

1. **テキストノードに触らない**
   - `<h1>` `<h2>` `<p>` `<li>` 等の文字列は変更禁止
   - クラス名・id・data 属性も変更禁止
2. **DOM 構造を変えない**
   - 要素の追加/削除/並び替えは禁止
   - 既存 `<img>` の差替えと、指定位置への `<img>` 新設のみ可
3. **スタイル変更は最小限**
   - 画像配置に必要な `width / height / object-fit / border-radius` のみ
   - 既存のレイアウトを崩さない範囲で
4. **画像形式とパス**
   - 配置先: `public/images/{section}/{name}.{ext}`
   - 形式: `.webp` を優先、フォールバックに `.jpg`
   - `next/image` を使い、`alt` 必須
5. **コミットは画像差替分を1セクション1コミット**
   - メッセージ例: `feat(lp): replace hero image with new salon photo`

## ページ別 差替シート（テンプレート）

各ページごとに以下のシートを1枚埋めて Claude Code に渡す。
**画像はオーナー提供。同じセクションで複数枚ある場合は番号で識別。**

### ── テンプレート ──

```markdown
## SECTION: {section-id} / {section-name}

### 差替対象
- **ファイル**: `app/_sections/{Component}.tsx` （または該当する jsx/html）
- **既存画像**: {既存のパス or 「なし（新設）」}
- **新画像ファイル名**: `{name}.webp`
- **配置パス**: `public/images/{section}/{name}.webp`
- **代替形式**: `public/images/{section}/{name}.jpg`（フォールバック）

### 画像仕様
- **被写体**: {例: 30代女性顧客が iPhone を見て微笑む / サロン店内 / ロゴ}
- **比率**: {例: 4:5 縦 / 1:1 正方形 / 16:9 横}
- **推奨実寸**: {例: 1200×1500px}
- **alt テキスト**: 「{alt 用日本語}」
- **角丸**: {例: rounded-2xl / 24px / なし}
- **オーバーレイ**: {例: 黒 30% グラデーション / なし}

### 配置位置（既存DOMのどこに入れるか）
{例:
- セクション右カラム、`<div class="phone-area">` の内側
- iPhone モック内 「style-card」 の上部
- または DOM パス: `.hero__right > .phone > .style-card > .style-card__img`
}

### 注意事項
- DOMの追加・削除は不可。既存の `<img>` を差替えるか、すでに用意されている空のスロット要素に挿入する。
- 既存の plain HTML プレースホルダー（CSS グラデーション背景）は `<img>` で置換する。
- レスポンシブ: `sizes="..."` を Next.js の規約に従って指定する。
```

## 進め方

1. オーナーがセクション N の写真をアップロード（命名は任意）
2. SalonRink 側で本リポジトリに `public/images/{section}/` に配置
3. 上記テンプレに沿った差替シート（`PATCH-{NN}.md`）を発行
4. Claude Code に差替シートを渡して該当セクションのみ書き換え
5. PR を立て、テキスト変更がないことを diff で確認
6. レビュー後マージ

## Claude Code への共通プロンプト（毎回これを使う）

```
SalonRink.com の LP に、画像のみを差替/新設してください。

【厳守】
- テキスト・配列・文章・クラス名・id は一切変更禁止
- DOM の追加/削除/並び替えは禁止
- 画像の差替え、または既存の空スロット要素への画像挿入のみ可
- スタイルは画像配置に必要最小限（width/height/object-fit/border-radius）のみ
- next/image を使う。alt は指示書に従う

【入力】
- 差替シート: PATCH-{NN}.md
- 画像ファイル: public/images/{section}/{name}.{webp,jpg}

【出力】
- 該当セクションの該当箇所のみ修正したコミット
- diff にテキスト変更が一切ないことを確認

完了後、git diff の要約と「テキスト変更ゼロ」の確認を報告してください。
```

## よくある質問

**Q. テキストの誤字も直してほしい場合は？**
A. このスコープでは対応しない。別タスク（コピー修正）として明確に依頼する。

**Q. 画像と一緒に CSS の角丸を変えたい**
A. 画像配置に必要な範囲（角丸・object-fit など）は可。それ以外の見た目変更は別タスク。

**Q. 既存画像のパスが分からない**
A. SalonRink リポジトリで `grep -r "{画像名}"` で特定。あるいは Claude Code に「該当箇所を特定して」と依頼。

**Q. アップロードする写真の命名規則は？**
A. 任意で OK。差替シート発行時に正式名を割り当てる。
