## 1. 推奨表現（OK）

### サービスの立ち位置を語るとき
- 「LINE公式アカウント拡張ツール」
- 「LINEに**乗せる**業務レイヤー」
- 「いつものLINEの中で完結」
- 「LINEを**活かしきる**」
- 「サロンの**業務システム**」

### 価値を語るとき
- 「**サロンの売上につながる**」
- 「お客様との関係を深め、サロンの**成長を加速**」
- 「**思い出す時間**を、**接客時間**に変える」
- 「**流れる会話**を、**資産**に変える」
- 「サロンの**価値が自然と積み上がる**」

### AIを指すとき
- 「**Concierge**（コンシェルジュ）」※サービス内の機能名としてのみ
- 「**AIが自動で**◯◯」
- 「**AIが書きおこす**」「**AIが提案**」

### スタッフ・オーナー向け表現
- 「面倒な作業から解放」
- 「接客に集中できる」
- 「朝のLINEで状況を把握」
- 「**売上速報 / リピート率 / 失客予兆**」

## 2. 廃止表現（NG）

旧 LP の主張のうち、新方針に合わない / ユーザー向けすぎるものは廃止する。

| ❌ NG | ⭕ 置換 |
|---|---|
| サロンに、専属コンシェルジュを。 | いまのLINE公式アカウントを、サロンの売上につながる業務システムに。 |
| お客様一人ひとりに、専属コンシェルジュを | （主語をオーナー / スタッフ / サロンへ） |
| 大手サロンの体験を、すべての店舗に | （導入規模を訴える文脈以外では使わない） |
| プロンプト / トークン / LLM / ベクトル検索 | （技術用語を表に出さない。「AIが自動で」「会話を読み取って」等に置換） |
| 「セキュリティはエンタープライズ級」 | 「あなたのサロンの情報を、しっかりお守りします」 |
| カルテ管理ツール / 予約管理ツール | LINEを活かす業務システム / サロン業務レイヤー |
| 解析 / アナリティクス | 状況を可視化 / 数字で把握 |

### トーン &amp; マナー
- **過度な敬語は避ける**（「〜いただけますでしょうか」より「〜ください」）
- **専門用語禁止**: API / SaaS / LLM / SDK 等は本文で使わない（料金ページなどスペック箇所は可）
- **絵文字**: コピー本文では使わない。**LINEトークモック内のみ可**
- **「！」の多用禁止**: 1セクションに1個まで
- **箇条書きは「。」で締めない**（タイトル相当の短いリスト項目）

## 8. 数値の管理

- 効果数値（−42% / +28% / +18% など）は、半年ごとに最新値へ更新可能とする
- 数値の根拠は **Numbers セクションのフットノート** に必ず明記
- 価格は税抜表示・別途消費税の表記が必要

数値変更時は **本辞書 §4 と各セクションHTMLの両方を更新**する。
